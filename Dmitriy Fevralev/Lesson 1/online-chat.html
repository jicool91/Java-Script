1. Форма для ввода текста в чате
  2. Принимает текст по нажатию кнопки "Отправить".
  3. Проверяет не пустое ли поле для ввода
  4. Отсекает все лишнее (trim,striptags,htmlspecialchar)
  5. Отправляет на сервер
  6. Сервер принимает текст из формы в переменную
  7. Переменная снова обрабатывается на корректность входящих данных
  8. Возвращает значение переменной клиенту
  9. а так же отправляет эти данные в виде переменной в sql запросе в БД для хранения истории
